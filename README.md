## spfxreactgrid

An SPFX Webpart that uses React, Office-UI-Fabric, and Redux to let users edit list data from multiple Webs and Multiple Sites in a single grid.

Build:
```bash
git clone the repo
npm i
gulp serve
```

1. Define the columns to be shown in the grid
